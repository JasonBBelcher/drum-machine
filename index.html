Open developer console to play with the sequence.

Edit the sequence by picking an index of the seq array that represents a tick of the sequencer. use the method signature like this: setState(drum, boolean, volumeLevel) drums available are:

  • kick
  • clap
  • snare
  • shaker
  • hat
  • perc
  • perc2

Things to try

Try turning individual drums on or off. Increase the tempo. Mess with the volume. Make an entirely new drum beat. Increase the sequence length from 8 to 16 

        
            let sequencer = new Sequencer(DrumMachineState, 8);
            let seq = sequencer.initSeq();
            seq[0].setState("kick", true, 1);
            seq[4].setState("kick", true, 0.8);
            seq[4].setState("clap", true, 0.5);
